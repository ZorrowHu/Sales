# Sales
A platform for brands and consumers to have free trades with vendors involved.

电商网站设计与实现——借卖平台项目

### 借卖平台项目介绍

* 管理员角色管理平台的账务与用户信息
 
* 品牌商管理商品仓库，并直接发货给用户

* 借卖方从品牌商仓库处把商品上架到店铺，并展示到商城首页，从中赚取差价

消费者最终在商城页面看到的是各个店铺上架的各个商品
